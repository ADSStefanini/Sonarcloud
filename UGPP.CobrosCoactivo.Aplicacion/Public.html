Software Coactivo S&P !

Esta solución de negocios esta basado sobre EIP (Enterprise Integration Portal) de manera que las compañías puedan enlazar todos sus puntos operativos a la central de datos financieros en tiempo real, sin importar el sitio del mundo en que se encuentre.

Tecnologías                                  

Microsoft .NET

Plataforma de desarrollo de software con énfasis en transparencia de redes, con independencia de plataforma de hardware y que permita un rápido desarrollo de aplicaciones.

Framework

La nueva tecnología de Microsoft ofrece soluciones a los problemas de programación enfocados al  desarrollar aplicaciones web, servicios XML y sitios web domésticos. Además de rutinas de idioma y librerías comunes, que forman también parte de la aplicación.

Microsoft SQL Server

Es el corazón y cerebro de la aplicación, Microsoft SQL Server es un sistema de gestión de bases de datos relacionales (SGBD) basado en el lenguaje Transact-SQL, y específicamente en Sybase IQ, capaz de poner a disposición de muchos usuarios grandes cantidades de datos de manera simultánea.
Permite trabajar en modo cliente-servidor, donde la información y datos se alojan en el servidor y las terminales o clientes de la red sólo acceden a la información.
